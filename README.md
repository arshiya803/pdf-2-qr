PDF to QR Code Management System
Overview

A full-stack web app to upload PDFs, generate QR codes, and share files publicly. Users can manage, download, delete and share their files easily.

Features

• Login/Register authentication
• PDF upload with QR generation
• My Files (view, download, delete)
• Public QR access without login
• Share via WhatsApp, Email, Telegram
• Dashboard with file stats
• Modern responsive UI

Tech Stack

Frontend: React.js
Backend: Node.js, Express.js
Database: MongoDB

Pages

Home | Dashboard | Generate QR | My Files | Share

How it works
Login/Register
Upload PDF
QR generated automatically
Share QR or link
Anyone can access file
